Apply the knowledge on using 'style tags' and using 'body' rule. 


Task:
Create a page that shows off a pallet of colors along with
their CSS names. 

 
Action:
HTML ->
*. Code the doctype, html, body elements.
1. Create an h1 heading titled, "Warhol Palette Example"
2. Create an h2 heading titled, "Deep Sky Blue"
3. Create an h3 heading titled, "Light Sky Blue"
4. Create an h4 heading titled, "Light Blue"
5. Create an image. Source,
   "https://mimo.app/r/mimowarhol.png"

CSS ->
1. Add the 'style' tags inside the 'head' tags.
2. Add an empty rule for the 'body' element.
3. To give the entire webpage some color,
   set 'background-color' to 'aliceBlue'.
4. Give the font a more familiar touch by setting 
   the 'font-family' to 'cursive'.
5. To style 'h2' element, make it's background to 'deepSkyBlue'.
6. Add empty rules for 'h3' and 'h4'.
7. Set the 'background-color' for 'h3' element to 'lightSkyBlue'.
8. Add the 'background-color' property for the 'h4' rule.

Situation:
In this project, practice the skills by styling a note taking app.
We'll use the app to note different Spanish words. 
The app uses different colors, fonts, and styles to keep track
of a user's progress. 


Task:
Part I
Use inline CSS to style the individual heading elements on the webpage.
 
Part II
Next, add 'style' tags and apply styles to multiple elements 
with just a few lines.


Action:
Part I
HTML ->
*. Code the doctype, html and head elements.
1. Code the h2 element titled "Spanish Vocabulary".
2. Add the first paragraph named, 
   "el abrigo, el lazo, la agujeta, el overol".
3. Add the second paragraph named, 
   "los huevos revueltos, el calzoncillo, los pantalones".
4. Add the third paragraph named,
   "los desarrolladores, los calcetines".
5. To mark the words that are easy to remember, code 
   the h3 heading titled, "Easy".
6. To mark the words that are known but might need a bit of
   practice, code the h3 heading titled, "Good".
7. To mark the words that need a lot of practice, code
   the h3 heading titled, "Hard".

CSS ->
1. To make the heading stand out, add 
   'style = "background-color: darkSeaGreen;"' to the first h3 element.
2. To make the heading stand out, add
   'style = "background-color: khaki;"' to the second h3 element.
3. To make h2 heading stand out, add
   'style = "background-color: pink;"' to the last h3 element.


Part II
HTML ->
1. Code the 'head' tags.
2. Inside the 'head' tags, add the 'style' tags.

CSS ->
1. Code a 'body' selector.
2. Inside the 'body' selector, set the 'background-color' to 'lightBlue'.
3. Code an 'h2' selector and inside it, set 'font-family' to 'Arial'.
4. Code an 'h3' selector and inside it, set 'font-family' to 'Courier New'.
5. Code a 'p' selector and set 'font-style' to 'italic'.
