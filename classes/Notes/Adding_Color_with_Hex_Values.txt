Let's use our knowledge of hex values to code a traffic light. 


Plan:
We added div elements with 'trafficLight', 'red', 'yellow' and 'green'
class names, to get you started

            <div class = "yellow"></div>
            <div class = "green"></div>
        </div>
    </body>
</html>

CSS Code ->
.trafficLight {
    height: 160px;
    padding: 10px;
} 


Action:
1. Set the background color of the traffic light using a hex color value.
2. Create a rule for all div elements to set margin-bottom to 5px,
   height to 50px, and width to 50px.
3. Let's get the first light up.
   Create a rule for elements of the '.red' class that sets 
   background color to #ff0000.
4. Create an empty rule for elements of the 'yellow' class.
5. For elements of the yellow class, set the background color to #ffff00.
6. Create a rule for elements of the green class that sets 
   background color to #00ff00.
7. Traffic lights should be round!
   Let's change all div elements by setting border radius to 25px.

Situation:
Let's code some flags using hex color values. 


Plan:
We'll code flags below each of the countries' names.
To get you started, we added some div elements and a bit of CSS. 


Availability:
HTML Code ->
<!doctype html>
<html>
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css">
    </head>
    <body>
        <div>
            <h3>Austria</h3>
            <p class = "red"></p>
            <p class = "white"></p>
            <p class = "red"></p>
        </div>
        <div>
            <h3>Gabon</h3>
            <p class = "green"></p>
            <p class = "yellow"></p>
            <p class = "blue"></p>
        </div>
        <div>
            <h3>Germany</h3>
            <p class = "black"></p>
            <p class = "red"></p>
            <p class = "yellow"></p>
        </div>
    </body>
</html>

CSS Code ->
body {
    background-color: #cccccc;
    text-align: center;
}

p {
    width: 200px;
    height: 50px;
    margin: 0 auto;
} 


Action:
1. For the elements of the red class,
   set the background-color to #ff0000.
2. For the elements of the white class,
   set the background-color to #ffffff.
3. Create a green class selector and set it's
   background color to #00ff00.
4. Create a yellow class selector and set it's
   background color to #ffff00.
5. Create a blue class selector and set it's
   background color to #0000ff.
6. Set the background color to #000000 for the 
   elements of the black class.
