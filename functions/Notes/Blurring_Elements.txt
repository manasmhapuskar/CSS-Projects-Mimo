Situation:
A new app, iCare, wants to teach users about eye health. 


Task:
They've asked us to create a bad vision simulation for their app. 


Given: 
HTML Code ->
<!doctype html>
<html>
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css" />
    </head>
    <body>
        <h1>Eye Sight Simulation</h1>
        <h3>VERY BAD</h3>
        <h4>Legally Blind</h4>
        <h5>Eat more carrots</h5>
        <h6>20/20</h6>
    </body>
</html> 


Action:
1. Start by adding a body selector.
2. Set the background color to beige.
3. As a next step, center the text.
4. Next, add an empty rule for h3 heading.
5. Add the 'filter' property and set it to 'blur()'.
6. To finish blurring the heading, add the value
   to 4px inside the parentheses.
7. To blur the h4 heading, code an h4 selector with
   filter property and blur set to 2px.
8. Use the h5 selector to blur h5 elements by 1px.

Situation:
A publishing company is trying out a new subscription model.


Task:
It needs your help blurring out some of it's content.  


Given:
HTML Code ->
<!doctype html>
<html>
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css" />
    </head>
    <body>
        <div>
            <h3>Baker News</h3>
            <h5>Tesla to change it's company name to Edison</h5>
            <button>Subscribe to Read More</button>
            <p>
                In a controversial move, Ellon Tusk has decided to change the
                company name of Tesla to Edison, while the newcomer car
                manufacturer Edison Destiny has decided to change it's name to
                Westinghouse.
            </p>
        </div>
        
        <h5>Can you guess the next week's wrestlemania contestants?</h5>
        <button>Subscribe to Find out More</button>
        <div>
            <img src = "https://mimo.app/i/darkmode.png" alt = "" title = "" />
            <img src = "https://mimo.app/i/ninjamode.png" alt = "" title = "" />
        </div>
    </body>
</html>

CSS Code ->
body {
    text-align: center;
}


Action:
1. As a first step, add an img selector.
2. Next, add width as 100px and height as 100px.
3. Set the image margins to 20px.
4. To tease the viewers about the content, 
   add 'filter' property and set its blur value to 10px
   to the images.
5. Create a p selector.
6. Set the font-size to 9px;
7. Add the blur effect with 10px value to tha paragraph.
