Problem:
Let's use the display property to change 
how elements appear and work on a sign-in form 


Available:
We'll start off with all of the form elements in place and
margins and paddings already set for input, button and div elements.
HTML Code ->
<!doctype html>
<html>
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css">
    </head>
    <body>
        <h1>Sign In</h1>
        <input placeholder = "Email">
        <input placeholder = "Password">
        <a href = "#">Forgot Password</a>
        <button>Log in</button>
        <div>facebook</div>
        <div>google</div>
        <div>linkedin</div>
    </body>
</html>

CSS Code ->
input, button {
    margin: 10px;
    padding: 5px;
}

div {
    margin: 5px;
    padding: 5px;
}


Solution:
1. Make the input and button elements begin on new lines
   by setting the 'display' property to 'block'.
2. Make both input and button elements have a width of 250px.
3. Make the div elements at the bottom adjustable in size and
   appear all in one line, set the 'display' property to 'inline-block'.
4. Style the div elements by setting the background color to plum.
5. To make "Forgot Password" link have a bigger surface area and
   be easier to click, set 'display' to 'block' for all link elements.
6. Set color to red for a elements.
7. Set the width of link elements to 250px.
8. Align the link by setting text align to center.

Problem:
Let's practice how to use display values by working on a website's footer,
the group of links at the bottom of the page.


Available:
We'll start off with the footer elements already added
inside a div element that has a few basics styles yet.
HTML Code ->
<!doctype html>
<html>
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css">
    </head>
    <body>
        <div>
            <h3>Footer Menu</h3>
            <a href = "#">Home</a>
            <a href = "#">Log in</a>
            <a href = "#">Sign up</a>
            <a href = "#">FAQ</a>
            <span>Copyright 2020</span>
        </div>
    </body>
</html>

CSS Code ->
div {
    background-color: lavender;
    width: 300px;
    padding: 10px;
    text-align: center;
}


Solution:
1. To make the lists adjustable in size,
   set the 'display' property of link elements to 'inline-block'.
2. Adjust the size of link elements by setting the width to 100px.
3. Make link elements stand out by setting background color to plum.
4. Add space between the links by setting margin to 10px.
5. Make the links bigger by setting padding to 10px.
6. To start styling the text "Copyright 2021", code a span selector,
   and inside it, set the 'display' property to 'block'.
7. Set the background color of the span element to purple.
8. Make the span element stand out by setting its color to snow.

Problem:
Let's use display values to style a social media post
that uses tags. 


Available:
We've added all the HTML and we've also given the div element
that makes up the social media post the class '.post' and styled it.
HTML Code ->
<!doctype html>
<html lang = "en">
    <head>
        <link rel = "stylesheet" type = "text/css" href = "style.css">
    </head>
    <body>
        <div class = "post">
            <em>posted by @degrasse 3 days ago</em>
            <h3>Pluto is not a planet, deal with it.</h3>
            <div class = "tag">planet</div>
            <div class = "tag">nasa</div>
            <div class = "tag">science</div>
            <a href = "#">3k views</a>
        </div>
    </body>
</html>

CSS Code ->
.post {
    width: 300px;
    border: 1px solid plum;
    padding: 10px;
}

em {
    color: gray;
} 


Solution:
1. Create a '.tag' class selector and 
   set the display property to 'inline'.
2. Set the background color of the '.tag' class to plum.
3. Set the padding elements of the '.tag' class to 5px.
4. Set the border radius of the '.tag' class to 10px.
5. Create an empty a selector.
6. Code the 'display' property and set it to 'block' to 
   make the link begin on a new line.
7. Code the margin top property and
   set it to 20px for the link element.
8. Adjust the color of the link so that they match
   the rest of the post by setting color to purple.
